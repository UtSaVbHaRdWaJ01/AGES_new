## 📋 About This Website

**AGES Premium Website** is a modern, professional, white-background website designed for Anshumala Green Energy Solutions — a Pan-India solar panel installation company based in Agra, Uttar Pradesh.

- **Design:** Clean, modern, premium aesthetic with white backgrounds
- **Colors:** Forest green (#0d7a3e), dark blue backgrounds for cards, royal blue accents
- **Fonts:** Playfair Display (serif headings) + Plus Jakarta Sans (body)
- **Features:** Fully responsive, animations on scroll, mobile-optimized

## 🚀 How to Use

1. Download and unzip the `AGES_Premium` folder
2. Double-click `index.html`
3. Opens in your browser — done!

### Option 2 — Host Online (Free)

**Netlify (Easiest)**
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop the `AGES_Premium` folder
3. Get a live website in seconds!

**GitHub Pages**
1. Create a GitHub repo
2. Upload the files
3. Enable Pages in repo settings
4. Live in minutes

**Hostinger / GoDaddy**
1. Upload files via cPanel File Manager
2. Point domain to folder
3. Done!

## 🎨 Customization Guide

### Change Colors

Open `css/style.css` and edit the `:root` variables (lines 6-20):

```css
:root {
  --G: #0d7a3e;    /* Green */
  --G2: #12a354;   /* Green hover */
  --G3: #16c963;   /* Bright green accents */
  --BL: #1565c0;   /* Blue */
  --B: #04142a;    /* Dark background */
  --W: #ffffff;    /* White */
}
```
